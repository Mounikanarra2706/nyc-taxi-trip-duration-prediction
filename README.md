# NYC TAXI TRIP DURATION PREDICTION

## Problem Statement
Pidicting the trip duration of NYC taxi rides using all regression models.

## Project goal
Building an end to end machine learning project by using PYSPARK for data analysis and preprocessing, and Prediction is done with Regression models.

## Tech Stack
- Python
- PySpark
- Scikit-learn
- Jypyter Notebook
- Vscode

## Project Structure
- data/: raw and processed data
- notebook/: EDA and experiments
- src/: reusable source code
- models/: saved models
- reports/: results and visualizations
